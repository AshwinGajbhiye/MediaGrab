# MediaGrab 🎬

MediaGrab is a lightning-fast, highly reliable web application for downloading Instagram Reels and YouTube Shorts/Videos in maximum quality without any watermarks or compression.

### 🚀 Live Demo
**Try it out here:** [https://mediagrab-e2xt.onrender.com/](https://mediagrab-e2xt.onrender.com/)

---

### Features
* **Instagram Support:** Download any public Instagram Reel instantly in raw 1080p format.
* **YouTube Support:** Download YouTube Shorts and standard YouTube videos in the highest available resolution (up to 4K).
* **No Compression:** Connects directly to RapidAPI scrapers to bypass CDN compression.
* **100% Private:** No user data is logged or tracked.

### Tech Stack
* **Frontend:** Vanilla HTML, CSS, JavaScript (Dynamic UI with loading states)
* **Backend:** Node.js, Express.js
* **Integrations:** RapidAPI (Instagram Scraper V2 & YouTube Media Downloader)
* **Hosting:** Render.com
